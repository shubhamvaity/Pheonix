<h3 align="center">A modern micro-blogging platform built using Django, HTML, CSS, & Javascript</h3>

------------------------------------------
### Features

- `Search for users/tweets`
- `Follow/Unfollow users`
- `Like/Dislike tweets`
- `Reply to tweets`
